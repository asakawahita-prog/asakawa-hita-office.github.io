<h1>朝川事務所</h1>

<!-- 事務所紹介 -->
<p>
  相続や遺言のことを、どこに相談すればいいのかわからない方が、
  安心して話せる行政書士事務所です。<br>
  大分県日田市を拠点としています。
</p>

<p>
  相続のこと、「まだ大丈夫」と思っていませんか。<br>
  実は、相続トラブルの多くは、始まる前から“兆し”が見えていることがあります。
</p>

<p>
  それでも、<br>
  ・何から考えればいいのか分からない<br>
  ・家族の関係を壊したくない<br>
  ・大ごとにしたくない<br>
  そう思って、つい後回しにしてしまう方が少なくありません。
</p>

<p>
  こんなお気持ちはありませんか。
</p>

<ul>
  <li>親の借金や保証の話を聞くと、胸がざわつく</li>
  <li>相続の話をすると、家族の空気が悪くなりそう</li>
  <li>何を聞けばいいのか分からず、先延ばしにしている</li>
  <li>子どもに面倒を残したくない</li>
</ul>

<p>
  ひとつでも当てはまる方は、<br>
  「今はまだ何もしない」ための整理が必要な段階かもしれません。
</p>

<!-- 業務内容 -->
<h2>業務内容</h2>
<ul>

</ul>
